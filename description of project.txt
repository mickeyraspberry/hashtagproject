Сбор данных по хэштегу vk.com (вывод статистики и графиков).
По файлам: 
access_tocken - файл для получения токена
graph - построение графиков, тут происходит основное действие по запросам к обновлению а так же реализация real time
ui - user interface - вот первое окно - это и есть маленькое окошечко, файл, с которого надо запускать
vkapi - отправление запросов в VK для получения информации

Начинаем искать хештеги для новостей, которые добавляются с текущего времени + 200 первых, которые были добавлены до этого времени

Технологии:
1. Tkinter для UI
2. MatPlotLib для графиков
3. API VK
4. Регулярные выражения
